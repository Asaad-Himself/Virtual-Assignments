SECTION A - DATA STRUCTURES

Strings

Strings are sequence of characters that are used to represent texts. They can include letters, numbers, symbols, and spaces. For example, "Hello, World!" and "12345" are both strings. Strings are usually enclosed in quotation marks, either single (' ') or double (" "), to denote the start and end of the string.

String Methods:

  1. length() - Returns the number of characters in a string.
  2. toUpperCase() - Converts all characters in a string to uppercase.
  3. toLowerCase() - Converts all characters in a string to lowercase.
  4. indexOf() - Returns the index of the first occurrence of a specified value in a string.
  5. substring() - Returns a part of a string between two specified indices.

Arrays

Arrays are used to store multiple values in a single variable. They can store elements of different data types, such as numbers, strings, or objects. Arrays are ordered collections of elements, and each element has a unique index starting from 0.

Array Methods:

  1. push() - Adds a new element to the end of an array.
  2. pop() - Removes the last element from an array.
  3. shift() - Removes the first element from an array.
  4. unshift() - Adds a new element to the beginning of an array.
  5. slice() - Returns a new array containing a portion of the original array.
